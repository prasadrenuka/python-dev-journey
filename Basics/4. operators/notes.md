Comparison Operators

| Operator | Meaning               | Example  | Result  |
|----------|-----------------------|----------|---------|
| `==`     | Equal to              | `5 == 5` | `True`  |
| `!=`     | Not equal             | `5 != 2` | `True`  |
| `>`      | Greater than          | `5 > 2`  | `True`  |
| `<`      | Less than             | `5 < 2`  | `False` |
| `>=`     | Greater than or equal | `5 >= 5` | `True`  |
| `<=`     | Less than or equal    | `5 <= 2` | `False` |

Assignment Operators

| Operator | Example   | Equivalent   |
|----------|-----------|--------------|
| `=`      | `x = 10`  | Assign value |
| `+=`     | `x += 5`  | `x = x + 5`  |
| `-=`     | `x -= 5`  | `x = x - 5`  |
| `*=`     | `x *= 5`  | `x = x * 5`  |
| `/=`     | `x /= 5`  | `x = x / 5`  |
| `//=`    | `x //= 5` | `x = x // 5` |
| `%=`     | `x %= 5`  | `x = x % 5`  |
| `**=`    | `x **= 5` | `x = x ** 5` |

Logical Operators

| Operator | Meaning           | Example          |
|----------|-------------------|------------------|
| `and`    | Both must be True | `True and False` |
| `or`     | At least one True | `True or False`  |
| `not`    | Reverse result    | `not True`       |

Membership Operators
| Operator | Meaning              | Example              |
| -------- | -------------------- | -------------------- |
| `in`     | Exists in collection | `'a' in 'apple'`     |
| `not in` | Does not exist       | `'z' not in 'apple'` |

Identity Operators
| Operator | Meaning           | Example      |
| -------- | ----------------- | ------------ |
| `is`     | Same object?      | `a is b`     |
| `is not` | Different object? | `a is not b` |
